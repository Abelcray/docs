# elasticsearch

## 配置

### hosts
|  `类型`  |  `是否必填`  |  `默认值`  |  `含义`  |
| ------- | ----------- | --------- | ------- |
| string数组  |    必填    |   无  | 发送日志至Elasticsearch的地址 |


### index

|  `类型`  |  `是否必填`  |  `默认值`  |  `含义`  |
| ------- | ----------- | --------- | ------- |
| string  |    必填    |   无  | 发送至Elasticsearch后，日志数据存储的index |

### username
|  `类型`  |  `是否必填`  |  `默认值`  |  `含义`  |
| ------- | ----------- | --------- | ------- |
| string  |    非必填    |   无  | 如果Elasticsearch配置了用户名密码验证，需要填写请求的用户名 |


### password
|  `类型`  |  `是否必填`  |  `默认值`  |  `含义`  |
| ------- | ----------- | --------- | ------- |
| string  |    非必填    |   无  | 如果Elasticsearch配置了用户名密码验证，需要填写请求的密码 |
